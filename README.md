#
A tool written in python to find subdomains of a target and generate a scope.


### Install
```
# Linux, Windows, MacOS
pip3 install -r requirements.txt
```

### Usage
```console
duty@f:~$ python3 subcat.py -d domain.tld

usage: subcat.py [-h] -d DOMAIN [-sc] [--scope SCOPE] [-t THREADS] [-v]
```
